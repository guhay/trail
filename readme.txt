1.数据预处理
    1.1.地图分块处理，利用百度地图API将经纬度坐标转变重百度地图的图形块坐标，具体教程如下：
    https://www.cnblogs.com/jz1108/archive/2011/07/02/2095376.html
  处理完的文件存放在data/map_block_data文件夹中，文件名中最后的数字代表地图缩放级别，例如res18代表地图的缩放等级为18级

    1.2.将原始的经纬度数据转换成地图块数据，存储在data/raw_data文件夹下，数据文件中的最后一列表示地图块

    1.3.以星期为单位将地图块数据串起来组成一条条轨迹，数据存储在data/trajectory中

    1.4 生成正负样本数据，数据存储在data/trajectory中，以+结尾的表示正样本，-结尾的表示负样本
